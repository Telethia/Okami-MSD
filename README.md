A tool to convert Okami's .MSD files into .txt files.

This tool was made with the November prototype in mind; however it should be compatible with .MSD files from the PS2 (and Wii?) versions. 
Currently incompatible with .MSDs from the HD version as it has two extra characters, however this will be implemented in the future.
The kanji set is incomplete as of this moment as I need to figure out which characters correspond to which bytes.
